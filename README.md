### Love Checker 

Love Checker is a fun and interactive Streamlit web application that calculates a love compatibility percentage between two names.
It uses predefined special cases, name-based logic, and animated UI effects to deliver entertaining results.

---> This project is created purely for entertainment purposes.<---

# Features:

- Love compatibility calculation

- Predefined special pairs & special cases

- Animated loading progress

- Special results for 100% matches

- Dynamic images based on result

- Funny & entertaining messages

- Clean, colorful Streamlit UI

# Tech Stack:

- Language: Python

- Framework: Streamlit

- Logic: Name-based ASCII comparison

- UI: Streamlit + Custom HTML/CSS

# Installation & Setup:
1. Clone the Repository
git clone https://github.com/your-username/Love-Checker.git
cd Love-Checker

2. Install Dependencies
pip install -r requirements.txt

3. Run the Application
streamlit run app.py

# How It Works:

1] Enter your name and your crush’s name

2] Click "Calculate Love"

The app:

- Checks special cases

- Calculates compatibility using name logic

- Displays animated results

- Enjoy fun messages and visuals
